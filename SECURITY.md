# Security Policy

This repository is public, but its package is under private incubation and is
not published to npm.

Report security concerns privately through GitHub security reporting or
directly to the repository maintainer.

Do not publish credentials, secrets or exploitable details in a public issue.

The package must not introduce:

- registry credentials;
- publication tokens;
- Atelier-Kit application secrets;
- client-site secrets;
- hidden network or telemetry dependencies.
